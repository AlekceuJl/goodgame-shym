# GoodGame PC Shym

Лендинг сервиса по скупке компьютеров и комплектующих в Шымкенте.

## Структура

```
goodgame/
├── index.html      ← разметка
├── styles.css      ← стили
├── robots.txt      ← правила индексации
├── sitemap.xml     ← карта сайта
└── images/         ← логотип, фото ПК, процессоров, видеокарт
```

## Запуск локально

```bash
python -m http.server 8000
```

Открыть http://localhost:8000
